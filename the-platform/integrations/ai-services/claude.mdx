### Description

Send AI chat messages using Anthropic Claude through Plura.

Use this integration when you want to use Claude models for high-quality, safe conversational AI.

### Authentication

- Configure your Anthropic API key in your Plura workspace settings.

### Example: Chat with Claude

```json
{
  "automationType": "ai_chat",
  "nodeId": "ai-node-004"
}
```
