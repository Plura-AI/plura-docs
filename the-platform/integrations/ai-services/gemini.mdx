### Description

Send AI chat messages using Google Gemini through Plura.

Use this integration to leverage Google’s Gemini models for chat, reasoning, and multimodal use cases.

### Authentication

- Configure your Google AI / Gemini API credentials in your Plura workspace settings.

### Example: Chat with Gemini

```json
{
  "automationType": "ai_chat",
  "nodeId": "ai-node-006"
}
```
