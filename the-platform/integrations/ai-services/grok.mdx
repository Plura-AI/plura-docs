### Description

Send AI chat messages using Grok (xAI) through Plura.

Use this integration when you want to power conversations with Grok’s real‑time, internet‑connected models.

### Authentication

- Configure your xAI / Grok API credentials in your Plura workspace settings.

### Example: Chat with Grok

```json
{
  "automationType": "ai_chat",
  "nodeId": "ai-node-005"
}
```
