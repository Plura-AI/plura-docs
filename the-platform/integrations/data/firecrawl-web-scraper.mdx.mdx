### Description

Use Firecrawl to scrape individual URLs or crawl entire websites and return clean, structured content (markdown, HTML, or extracted fields). Feed the results into AI agents, workflows, or data pipelines.

### Authentication

- Firecrawl API Key.

### Available Actions

- Scrape a single URL and return page content
- Crawl a site and collect content from matching pages
- Extract structured data using a target schema
- Map a domain's URLs for downstream processing
