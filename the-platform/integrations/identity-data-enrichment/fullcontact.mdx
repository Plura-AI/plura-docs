**Description:** Manage operations in FullContact.

**Available Actions:**

**Examples:**

##### Action: `unknown`

**Example 1: FullContact - Person Enrich**

Enrich a person by email/phone/profile. Docs: https://docs.fullcontact.com/docs/getting-started

```json
{
  "automationType": "fullcontact",
  "nodeId": "fullcontact-person-enrich-001"
}
```

**Example 2: FullContact - Company Enrich**

Enrich a company by domain or name. Docs: https://docs.fullcontact.com/docs/getting-started

```json
{
  "automationType": "fullcontact",
  "nodeId": "fullcontact-company-enrich-001"
}
```

**Example 3: FullContact - Identity Resolve**

Resolve identity using multiple fields (email, phone, name, address). Docs: https://docs.fullcontact.com/docs/getting-started

```json
{
  "automationType": "fullcontact",
  "nodeId": "fullcontact-identity-resolve-001"
}
```

**Example 4: FullContact - Stats Get**

```json
}
```

**Example 5: FullContact - Verification Email (v2)**
