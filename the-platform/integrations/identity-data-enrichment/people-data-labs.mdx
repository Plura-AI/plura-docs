**Available Actions:**

**Examples:**

##### Action: `unknown`

**Example 1: PDL - Person Enrich**

Enrich a person by email/phone/profile. Docs: https://docs.peopledatalabs.com/docs/person-enrichment-api

```json
{
  "automationType": "peopledatalabs",
  "nodeId": "pdl-person-enrich-001"
}
```

**Example 2: PDL - Person Search**

Search persons using Elasticsearch-style DSL. Docs: https://docs.peopledatalabs.com/docs/person-search-api

```json

  "nodeId": "pdl-person-search-001"
}
```
