**Description:** Manage operations in Jira.

**Authentication:**
- See integration-specific authentication requirements

**Available Actions:**

**Examples:**

##### Action: `unknown`

**Example 1: Jira - Create Issue**

Create a new Jira issue (task, bug, feature, etc.). Supports both API token and OAuth authentication.

```json
{
  "automationType": "jira",
  "nodeId": "jira-create-issue-001"
}
```

**Example 2: Jira - Create Issue (OAuth)**

Create a new Jira issue using OAuth authentication. Instance URL is retrieved from OAuth token metadata.

```json
{
  "automationType": "jira",
  "nodeId": "jira-create-issue-oauth-001"
}
```

**Example 3: Jira - Update Issue Status**
